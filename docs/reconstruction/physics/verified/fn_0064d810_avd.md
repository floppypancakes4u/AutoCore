## 4. Authoritative formula

```
w   = (rb.ωx, rb.ωy, rb.ωz)           // +0x50/+0x54/+0x58
w2  = wx² + wy² + wz²
thr = this.collisionThreshold         // +0x10

if w2 <= thr * thr:                   // |w| ≤ threshold  (equality → normal)
    d = this.normalSpinDamping        // +0x08
else:
    d = this.collisionSpinDamping     // +0x0c

f = max(0, 1 − d * dt)                // g_flOne − d*dt, then floor at 0

rb.ωx  *= f
rb.ωy  *= f
rb.ωz  *= f
rb.ωw  *= f                           // +0x5c also scaled
rb.setAngularVelocity(ω')             // vtbl +0x54
```

Closed form:

```
ω' = ω · max(0, 1 − d · dt)

d = (|ω| ≤ AVDCollisionThreshold)
      ? AVDNormalSpinDamping
      : AVDCollisionSpinDamping
```

where `|ω| ≤ thr` is implemented as **`|ω|² ≤ thr²`** (no `sqrt`).

### Branch semantics

| Condition | Rate used |
|-----------|-----------|
| `|ω| ≤ collisionThreshold` | **normal** spin damping (gentle cruise) |
| `|ω| > collisionThreshold` | **collision** spin damping (harder, speed-triggered) |

- Gate is **purely angular-speed-triggered**. No collision timer, no contact flag, no 6400 ms window.
- Equality at the threshold takes the **normal** branch (`<=`).
- If `d·dt ≥ 1`, scale becomes **0** → angular velocity fully zeroed this step.
- All three spatial axes + the 4th RB angVel slot share the **same** scalar `f`.
