## Purpose

Handle S2C `CharacterLevel` (`0x2017`): absolute money/XP/level snapshot onto the TFID object, then local UI refresh.

## Behavioral summary

1. Lookup object by packet TFID (`+8` lo, `+0xc` hi, `+0x10` global).
2. If found, call vfunc `+0xcc` → `CVOGCharacter_ApplyCharacterLevelPacket`.
3. If TFID matches local player (`game+0xe98`): `Client_RefreshLocalCharacterLevelUi` + optional HUD.
4. Refresh mission UI windows; optional secondary panel `game+0x10b0`.
